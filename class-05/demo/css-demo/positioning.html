CSS Positioning

Position: relative

It will display exactly the same as before (position static) until you also assign its position using: left, right, top or bottom. This will position it that distance relative to its original position.

Position: absolute

This element displays relative to its most recent parent element with positioning set. If it has no positioned parent, it will display relative to the page window.

Position: Fixed

This will position the element relative to the viewport window. It will stay in the same place even if the page scrolls. Be careful! It can block content on your page unless you account for it by adding padding or margin to other elements.
